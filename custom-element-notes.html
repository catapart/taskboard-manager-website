Notes on developing custom elements

Intro

This is a subpage of the <taskboard-manager> custom element's showcase page.

This page is a collection of notes about the development of custom elements for browsers. It is intended to be a guide for best practices, rather than a set of rules to follow.

Many of the conclusions drawn here may be naive or impractical for a lot of use cases. This page is mainly a reference.
